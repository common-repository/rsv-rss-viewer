RSV RSS is plugin for placing RSS feeds in your site.

== Description ==

RSV RSS is plugin for placing RSS feeds in your site. It will produce clean HTML code without any iframe.

== Installation ==

1. Upload the <strong>rsv-rss</strong> folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the '<strong>Plugins</strong>' menu in WordPress
3. Use This Short Code to Display RSS feeds.
4. You can change <strong>id</strong> for every shortcode used. 

== Frequently Asked Questions ==

= How do I insert the RSV RSS? =
You can insert the RSV RSS by pasting the shortcode into the content area of a page or post.  **Be sure to use the HTML editor when inserting shortcodes!** 

= where can find the 'Shortcode' ? =
Go to Rsv RSS in your WordPress admin, add RSS URL. Then you can see shortcode of that feed. Place that shortcode into the content area of a page or post.
